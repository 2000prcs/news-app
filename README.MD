An `nvmrc` file is included if using [nvm](https://github.com/creationix/nvm).

## Usage

From the root directory:

1. Install project dependencies & Build client bundles:

```sh
npm install
```

2. Start server:

```sh
npm start
```

4. Open browser at `127.0.0.1:3000` 

5. Enjoy!

## Testing

```sh
npm test
```
